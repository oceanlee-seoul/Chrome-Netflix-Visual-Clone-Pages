# 구글 크롬 및 넷플릭스 반응형 클론 페이지

⭐️ 개발 배경 및 목적

  2. 기초적인 jQuery를 사용하여 페이지 제작 연습
  3. 웹브라우저의 사이즈에 따른 반응형 웹페이지 제작 연습
